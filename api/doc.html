Het BotenInschijfSysteem biedt een eenvoudige RESTful API voor bijvoorbeeld gebruik door apps.

Op api/index.php kunnen twee soorten requests worden uitgevoerd: GET en POST.

Met GET kunnen de diverse database-tabellen (entiteiten) worden uitgevraagd. Voorbeeld: http://www.hunze.nl/test_bis/api/index.php?entity=types

De uit te vragen tabellen zijn:

  • bestuursleden
  • boten
  • inschrijvingen
  • inschrijvingen_oud
  • mededelingen
  • roeigraden
  • types
  • uitdevaart

De gegevens worden als key-value-paren in JSON-formaat teruggegeven. Voorbeeld: {"ID":"1","Type":"1x","Categorie":"Skiffs en C1en","Roeisoort":"scull"}

Met een POST-request kunt u een inschrijving aan de database proberen toe te voegen.

In de toekomst dient elk request vergezeld te gaan van de juiste authenticatie.
